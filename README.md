Overview

This project is an interactive, macro-enabled Excel dashboard that analyzes and compares European countries using crime statistics and key quality-of-life indicators. The goal is to support data-driven insights into safety, affordability, and overall livability across countries.

Key Features

Integrated datasets covering crime, cost of living, unemployment, dining, and public transportation

Interactive dashboard with charts and heat maps for easy comparison

Country-level analysis to identify patterns, trends, and outliers

Clean data structure with separate analysis sheets and a combined master dataset

Tools & Technologies

Microsoft Excel (Advanced formulas, Pivot Tables, Charts)

VBA (for interactivity and automation)

Data visualization using heat maps and dashboards

Use Cases

Comparative country analysis

Quality-of-life and safety assessment

Academic projects and portfolio demonstration

Business and consulting-style analytical reporting

Data Sources

All datasets used in this project are publicly available and cited within the workbook.
